Text Styling Examples

Example 1:

body { color: red; }

Output: This is red text.

Example 2:

p { font-size: 30px; }

Output:

This is big text.

Example 3:

h1 { font-weight: bold; }

Output: This is bold text.

Example 4:

span { text-decoration: underline; }

Output: This is underlined text.

Example 5:

p { font-style: italic; }

Output: This is italic text.

Example 6:

h3 { letter-spacing: 4px; }

Output: This is spaced text.
